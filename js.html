<Glossário Interativo de Tags JavaScript>


O que é a estrutura if?

Executa um bloco de código se uma condição for verdadeira.

Exemplo:

  int idade = 18;
  if (idade >= 18) {
      System.out.println("Maior de idade");
  }
O que é a estrutura else?

Executa um bloco de código caso a condição do if seja falsa.

Exemplo:

  int idade = 16;
  if (idade >= 18) {
      System.out.println("Maior de idade");
  } else {
      System.out.println("Menor de idade");
  }
O que é a estrutura else if?

Permite testar múltiplas condições sequenciais.

Exemplo:

  int nota = 75;
  if (nota >= 90) {
      System.out.println("Excelente");
  } else if (nota >= 70) {
      System.out.println("Bom");
  } else {
      System.out.println("Precisa melhorar");
  }
O que é a estrutura switch?

Seleciona um bloco de código para executar baseado no valor de uma variável.

Exemplo:

  int dia = 3;
  switch (dia) {
      case 1:
          System.out.println("Domingo");
          break;
      case 2:
          System.out.println("Segunda");
          break;
      case 3:
          System.out.println("Terça");
          break;
      default:
          System.out.println("Outro dia");
  }
O que é a estrutura case?

Define cada condição dentro de um switch.

Exemplo:

  switch (opcao) {
      case 1:
          System.out.println("Opção 1");
          break;
      case 2:
          System.out.println("Opção 2");
          break;
      default:
          System.out.println("Opção inválida");
  }
