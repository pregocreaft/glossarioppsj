<Glossário Interativo de Tags CSS>


O que é a propriedade color?

Define a cor do texto.

Exemplo:

p { color: silver; }
O que é a propriedade background-color?

Define a cor de fundo de um elemento.

Exemplo:

div { background-color: black; }
O que é a propriedade font-size?

Define o tamanho da fonte do texto.

Exemplo:

p { font-size: 16px; }
O que é a propriedade margin?

Define a margem externa de um elemento.

Exemplo:

div { margin: 20px; }
O que é a propriedade padding?

Define o espaçamento interno entre o conteúdo e a borda do elemento.

Exemplo:

div { padding: 10px; }
O que é a propriedade border?

Define a borda de um elemento.

Exemplo:

div { border: 2px solid silver; }
O que é a propriedade display?

Define como um elemento será exibido (bloco, inline, flex, etc.).

Exemplo:

div { display: flex; }
O que é a propriedade text-align?

Define o alinhamento horizontal do texto dentro de um elemento.

Exemplo:

p { text-align: center; }
O que é a propriedade width?

Define a largura de um elemento.

Exemplo:

div { width: 200px; }
O que é a propriedade height?

Define a altura de um elemento.

Exemplo:

div { height: 150px; }
