<Glossário Interativo de Tags HTML>


O que é a tag <link>?

Usada para vincular arquivos externos, como CSS.

Exemplo:

<link rel="stylesheet" href="style.css">
O que é a tag <div>?

Cria um bloco para agrupar elementos.

Exemplo:

<div class="caixa">Conteúdo</div>
O que é a tag <p>?

Define um parágrafo de texto.

Exemplo:

<p>Este é um parágrafo.</p>
O que é a tag <br>?

Insere uma quebra de linha.

Exemplo:

Linha 1<br>Linha 2
O que é a tag <hr>?

Insere uma linha horizontal.

Exemplo:

<hr>
O que é a tag <a>?

Cria links para páginas ou recursos.

Exemplo:

<a href="index.html">Voltar</a>
O que é a tag <button>?

Cria um botão clicável.

Exemplo:

<button>Enviar</button>
O que é a tag <img>?

Insere imagens na página.

Exemplo:

<img src="foto.png" alt="Descrição">
O que é a tag <span>?

Permite aplicar estilos em partes do texto.

Exemplo:

<p>Texto <span style="color:silver">prata</span>.</p>
O que é a tag <footer>?

Define o rodapé da página.

Exemplo:

<footer>Todos os direitos reservados</footer>
